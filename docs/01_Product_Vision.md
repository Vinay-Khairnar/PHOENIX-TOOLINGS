# Product Vision

## Goal
QuoteMate (formerly QuotationOS) is a lightweight quotation generator designed for businesses that receive supplier price lists in PDF or CSV format.
The application helps users quickly search products, enter quantities, apply discounts, and generate professional quotations without manual calculations.

## Target Users
Small businesses, Retailers, Wholesalers, Manufacturers, Tool Rooms, Medical Suppliers.

## Core Features
- Import product list
- Search products
- Create quotations
- Apply discounts
- Export quotation as PDF

## Design Principles
Simple, Fast, Offline-first, Minimal, Reliable, Easy to maintain.

## Success Metric
A complete quotation should be generated in less than one minute with zero manual calculations.
