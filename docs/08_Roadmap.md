# Roadmap

## Phase 1: MVP (Current)
- Basic SQLite schema.
- CSV import for products.
- Quotation builder UI.
- PDF generation via pdf-lib.

## Phase 2: Refinement
- Enhanced search performance (SQLite FTS5).
- Better PWA offline caching strategies.
- Settings management for customizable PDF headers.

## Phase 3: Polish
- Localization / currency formatting options.
- Dark mode support aligned with Apple's dark tile aesthetic.
