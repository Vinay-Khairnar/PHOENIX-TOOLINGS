# Rules

1. **Keep it Simple**: No feature creep. No CRM, no inventory, no complex roles.
2. **Apple-style UI**: Edge-to-edge layouts, high contrast, Action Blue (`#0066cc`), SF Pro (Inter), minimal decorative UI chrome. No gradients or box-shadows on cards.
3. **Speed is King**: The entire workflow from opening the app to generating a PDF must take < 30 seconds.
4. **Local First**: SQLite database keeps everything fast and contained.
5. **PDFs Must Look Professional**: Clean typography, clear tables, minimal branding.
